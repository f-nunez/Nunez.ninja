Description:

Veterinary Clinic management system made in NET 7. With clean architecture.

The Veterinary Clinic System consists of two web applications. The Clinic Management app to manage clinics, doctors, clients and patients. And the Scheduling app to manage Patient's appointments.

Backend Technologies:

ASP.NET Core 7, AutoMapper, Entity Framework Core 7, FluentValidation, IdentityServer, MassTransit, MediatR, RabbitMQ, SignalR, SQL Server, YARP.

Frontend Technologies:

BFF (Backend for Frontend) Security Framework, Blazor, Radzen Components.
